# M4-PF-Assignment
Andela GenAI Fourth Assignment


## Setup

Command to setup database:
```bash
python src/database/init_db.py
```

Command to run the application:
```bash
python src/main.py
```

Command to quit the application:
Type 'exit', 'quit', or 'q' to end the program.
